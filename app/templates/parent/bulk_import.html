📥 Bulk Import NCERT Books

Import all chapters from a structured book directory

ℹ️ Only new chapters (not already in SchoolBell) will be imported. Existing chapters are left untouched. Books stay in the source folder.

How to add books

Drop your grade folders into the project's ncert_books/ folder: 

ncert_books/          ← inside the SchoolBell project
  Grade_8/
    Science/
      chapter01.pdf
      chapter02.pdf
    Maths/
      chapter01.pdf

Grade number is read from the folder name (e.g. Grade_8). Chapter title & number are extracted from the first page of each PDF.

Type any folder path, or reset to project default .
